# PoC-HideSys-Driver
PoC de um driver que usa sua liberdade no espaço do Kernel para interceptar funções de leitura e ocultar drivers maliciosos.

**Driver criado para fins educacionais, não me responsabilizo por uso indevido. Para se proteger contra algo desse tipo, mantenha seus sistemas sempre atualizados e o antivírus sempre ativo.**
